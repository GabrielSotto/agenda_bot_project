# AGENDA BOT

Este proyecto organiza mis días automáticamente
y me manda la lista de tareas a Telegram, según sean días libre, días laborales, también categoriza, prioriza tareas, elimina teas completadas y reagenda las no completadas para el siguiente dia libre. Todo esto lo hace leyendo mis hoarios laborales del mes desde la grilla que me dan, asigna tareas que están en una base de datos de tareas y teniendo en cuanta mis horarios laborales rotativos

Estructura:
- main.py: Código principal.
- config.json: Configuraci¢n de claves.
- turnos.csv: Horarios de trabajo.
- tareas.csv: Lista de tareas pendientes.
- prueba_verificacion.py -> Sistema de verificación automática de archivos y configuraciones.
